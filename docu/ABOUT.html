About Quality Cat Game Library

Quality Cat Game Library (name pending) is a small, lightweight open-source JavaScript library for developing games with the HTML5 canvas element. At this time, MANY of its features are still work-in-progress, and some are even unstable.

Why the name?

Branding, pretty much.

ANOTHER Library?!

Yes, another library. I didn't want to deal with the steep learning curve and massive overhead of using most of the other libraries out there. I also didn't feel like paying money for access to most of the engines out there, like GameMaker's HTML5 module and the full version of Construct2. I also found it pretty ironic/annoying that the supposed "open web" is being milked by the various groups and individuals developing paid, proprietary engines and IDES.

What can it do?

  1. Make the HTML Canvas element a LOT easier to use
  2. Create custom user interfaces (WIP, BUGGY!)
  3. Manage assets and asset loading(WIP)
  4. Manage keyboard and mouse input

What can't do?

  1. Replace highly advanced IDES/Libraries/Engines like Flash,Unity,CreateJS,Jquery
  2. playing audio, HTML5 handles this poorly, in general. (planned?)
  3. throw an entire game/engine/'app' together in almost no time at all
  4. achieve real-time performance (cuz, JavaScript!)
