core.Game

A basic game class

properties

  • state:String, the current state, defaults to "standby"
  • timing:Number, the timing/framerate, defaults to 33, measured in milliseconds

methods

  • Game():Game, constructor
  • setFPS(fps):void, sets the timing property to 1000/fps and rounds it to the nearest integer
  • init(canvasID):void, sets state to "initalize" and grabs the canvas with ID canvasID and adds that as the canvas property
  • tick():void, update/tick method, empty by default
  • draw():void, draw/render method, empty by default
